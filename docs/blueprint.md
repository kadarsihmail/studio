# **App Name**: DosenCheck

## Core Features:

- QR Code Generation: Generate a unique QR code for each lecture session.
- Attendance Scanning: Scan QR codes using mobile devices to record attendance.
- Attendance Tracking: Log the timestamp of QR code scans. For later reconciliation and analysis.
- Data Reporting: Allow user to export the attendence record by course, student and date range

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a professional and trustworthy feel.
- Background color: Light grey (#F5F5F5), to ensure high contrast with content.
- Accent color: Cyan (#00BCD4), drawing attention to interactive elements such as buttons and QR codes.
- Body and headline font: 'PT Sans' for clear readability and a modern aesthetic.
- Use clear, recognizable icons for navigation and key functions.
- A clean, intuitive layout for easy navigation, emphasizing key information and attendance actions.
- Subtle animations and transitions to give feedback, especially when attendance is recorded.
